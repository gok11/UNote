<h1 align="center">UNote</h1>

[![license](https://img.shields.io/badge/LICENSE-MIT-green.svg)](LICENSE.md)

[English Documents(英語ドキュメント)](README.md)

UNote は Unity 上でメモを作成・編集するためのツールです。
メモにはオブジェクトの参照や画像などの追加情報を持たせることができ、1つのメモに対して複数ユーザーから書き込むこともできます。

<img width="70%" src="Doc/readme_01.png" alt="01">

<h2>機能</h2>

* メモエディター: メモの追加/削除/編集
* Inspector編集: アセットを対象としたメモは Inspector からも編集可能
* メモプロパティ: タグで分類、お気に入りで優先表示、アーカイブで編集や表示を制限
* 追加情報の埋め込み: アセット参照等の情報をメモに追加可能
* スクリーンショット: スクリーンショットを撮影して追加情報として埋め込み。簡易画像エディターで編集可能
* 検索: テキスト内容やメモプロパティでのフィルタリング、表示順のソート。検索条件は自分で定義可能

追加予定の機能の概要については [TODO.md](Doc/Ja/TODO.md) を確認してください。

<h2>インストール</h2>

Unity2022.3 以降で動作します。

1. メニューから Window > Package Manager を選択
2. \+ ボタンを押して Add package from git URL... を選択
3. `https://github.com/gok11/UNote.git?path=/Assets/UNote` を入力して Add を押す

<h2>クイックスタート</h2>

<h3>ユーザー名の設定</h3>

まず自分のユーザー名を設定します。メモの編集権限にも影響するため複数人で利用する場合は重複を避け、最初に設定した名前は変更しないよう注意してください。

初期状態ではPCのユーザー名が設定されます。

1. メニューの Edit > Preferences... を選択
2. UNote > Editor Name を任意の名前に設定

<img width="70%" src="Doc/readme_02.png" alt="01">

<h3>プロジェクトメモとメッセージの追加</h3>

1. メニューバーから UNote > Note Editor を選択
2. 左下の Add Note... からメモ追加ウィンドウを表示
3. メモの種類を Project にして Add New Note を選択
4. 右ペインの上部からメモ名を変更
5. 右ペインの入力フィールドにテキストを入力して > ボタンかショートカットからメッセージを追加

自分で追加したメモ名やメッセージは後から編集できます。

<h3>アセットメモの追加</h3>

1. メモを追加したいアセットを選択
2. 入力フィールドからメッセージを追加

プロジェクトメモ同様 Note Editor から追加することもできますが、Inspector から追加する方が手順が少ないためオススメです。

<h3>スクリーンショット</h3>

右ペインの入力フィールドにある + ボタン > Screenshot > GameView を選択するとスクリーンショット撮影を行われ、入力フィールドに参照情報が追加されます。

メッセージを送信後、表示された画像にマウスオーバーして表示されるボタンを押すと簡易編集を行えます。

<h3>追加情報の設定</h3>

入力フィールドにあるタグアイコン > 任意のタグ選択でメッセージにタグを追加できます。

また、メッセージに参照情報を追加したい場合は対象のアセットを入力フィールドにドロップして追加できます。

<h3>検索</h3>

クエリを定義し、その内容に当てはまるメモのみを一覧に表示できます。

Custom Query の右のボタンを押すとクエリを追加できます。

クエリには以下の内容を指定できます。

* Search Term: メモ名かメッセージ内にこの文字列を含むか
* Type: メモの種類が等しいか
* Tag: メモに紐づくメッセージのいずれかがタグを含むか。None は全てのメッセージにタグがないメモを表示
* Sort: ソート方法。更新日時か作成日時が新しい/古い順に表示
* Display Archived: アーカイブしたメモも表示するか

現在のクエリを保存する場合は Save を選択します。
